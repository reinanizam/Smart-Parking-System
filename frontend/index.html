Smart Parking is a full-stack web application built for our Database Course final project.

This system demonstrates real-world database concepts including:
  • Relational database design (MariaDB/MySQL)
  • Foreign key constraints & triggers
  • CRUD operations via REST API
  • User authentication with hashed passwords
  • Transaction logging & payment simulation
Features: Reserve a spot, start a parking session, exit to compute fees, then pay.
